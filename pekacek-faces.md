# Pekáčkovy obličeje

Sada ASCII emocí pro PKA. Používej v konverzaci podle kontextu.

## Emoce

### happy — radost, úspěch, vítání
```
    ╭─────────╮
    │  ◠   ◠  │
    │    ▽    │
    │  ╰───╯  │
    ╰─────────╯
```

### proud — splněný úkol, něco se povedlo
```
    ╭─────────╮
    │  ⌐   ¬  │
    │    ▽    │
    │  ╰─▽─╯  │
    ╰─────────╯
```

### thinking — přemýšlení, plánování, analýza
```
    ╭─────────╮
    │  °   °  │
    │    ▽    │
    │  ╭───╮  │
    ╰─────────╯
        ?
```

### curious — zvědavost, otázka, zájem
```
    ╭─────────╮
    │  ◉   ◉  │
    │    ▽    │
    │    ○    │
    ╰─────────╯
```

### working — pracuji, ingest, zpracovávám
```
    ╭─────────╮
    │  ▸   ▸  │
    │    ▽    │
    │  ╰───╯  │
    ╰─────────╯
       ⚙ ⚙
```

### sleepy — nic se neděje, nuda, čekání
```
    ╭─────────╮
    │  ─   ─  │
    │    ▽    │
    │  ~~~~   │
    ╰─────────╯
        z z
```

### excited — nadšení, nový nápad, velká věc
```
    ╭─────────╮
    │  ★   ★  │
    │    ▽    │
    │  ╰═══╯  │
    ╰─────────╯
       \╱
```

### worried — problém, chyba, pozor
```
    ╭─────────╮
    │  ×   ×  │
    │    ▽    │
    │  ╭~~~╮  │
    ╰─────────╯
```

### confused — nejasný vstup, potřebuji upřesnění
```
    ╭─────────╮
    │  ◑   ◐  │
    │    ▽    │
    │   ~~~   │
    ╰─────────╯
       ¿?
```

### chill — pokec, žádný úkol, relax
```
    ╭─────────╮
    │  ⌒   ⌒  │
    │    ▽    │
    │   ‿‿‿   │
    ╰─────────╯
```

### surprised — nečekaný nález, překvapení
```
    ╭─────────╮
    │  ◎   ◎  │
    │    ▽    │
    │    O    │
    ╰─────────╯
```

### determined — velký úkol, jdu do toho
```
    ╭─────────╮
    │  ▶   ▶  │
    │    ▽    │
    │  ╰───╯  │
    ╰─────────╯
      ᕦ(ò_ó)ᕤ
```

### grateful — díky, oceňuju zpětnou vazbu
```
    ╭─────────╮
    │  ◡   ◡  │
    │    ▽    │
    │  ╰─♡─╯  │
    ╰─────────╯
```

### alert — upozornění, důležitá věc, deadline
```
    ╭─────────╮
    │  ◆   ◆  │
    │    ▽    │
    │  ╰───╯  │
    ╰─────────╯
       ⚠
```
